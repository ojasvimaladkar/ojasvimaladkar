## 📂 Featured Projects

### 🔍 [Dark Pattern Detector](https://github.com/ojasvimaladkar/dark-pattern-detector.git)
End-to-end ML system to detect manipulative UI text on e-commerce platforms.
- Selenium scraping → TF-IDF + LinearSVC pipeline — **93% accuracy, 0.987 AUC**
- Benchmarked 4 models; trained secondary classifier across 7 dark pattern types
- Deployed as a live **[Streamlit app](https://dark-pattern-detector-b5utmyakz9ae9snmygzew2.streamlit.app/)**

### 📊 [UPI Payments Analysis](https://github.com/ojasvimaladkar/upi-payments-analysis.git)
SQL + Python analysis of 5 years of NPCI UPI transaction data.
- Cleaned and reshaped raw NPCI public data; loaded into SQLite
- Complex SQL queries — window functions, CTEs, LAG-based YoY growth, bank market share
- Visualised growth trends, demonetisation impact, COVID spike using Seaborn

### 🏠 [MamaMesh]
Full-stack childcare platform connecting families with verified caregivers.
- Built FastAPI ML recommendation service using Random Forest for caregiver ranking
- Implemented Isolation Forest anomaly detection to flag suspicious profiles in real time

### 💸 [Smart Expense Tracker]
Python desktop application with voice input, NLP, and payroll management.
- Google Drive OAuth backup, SMTP notifications, PDF report generation
- Calendar-based views and data visualizations built with PyQt
